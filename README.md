#  SoulTrack – A Journey Within  

**SoulTrack** is a personal growth and reflection app designed to help users **track their emotions, thoughts, and daily experiences**. It combines mindful journaling with data-driven insights, allowing you to discover patterns in your mood, habits, and overall well-being.  
---
## ✨ Features  
- 📝 **Daily Journaling** – Log your thoughts and feelings with mood/emotion tagging.  
- 📊 **Mood Analytics** – Gain visual insights into your patterns and trends.  
- ⏳ **Streaks & Reminders** – Stay consistent with gentle nudges.  
- 🔒 **Secure Setup** – Environment variables (`.env`) for API keys and secrets.  
- 🌐 **API Ready** – Scalable backend to support mobile/web integration.
